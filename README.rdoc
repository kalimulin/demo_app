Приложение построено по книге "Гибкая разработка веб-приложений в среде Rails" 
(http://www.ozon.ru/context/detail/id/8153111/), по ней построен тестовый интернет-магазин по адресу: http://ror-store.tk 
Для работы приложения был арендован VPS и самостоятельно настроен веб-сервер Nginx+MySQL. 
Интернет-магазин имеет базовый функционал по добавлению/изменению/просмотру/удалению товаров, заказов и пользователей. 
Имеется административная зона для управления интернет-магазином. Приложение использует библиотеки jQuery и CarrierWave.
